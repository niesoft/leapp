# Public Functions

leapp.windowTitle("title");

Sets the name of the parent window

leapp.width = 500;

Sets the width of the parent window

leapp.height = 500;

Sets the height of the parent window

leapp.reload();

Reloads the initial screen of the window

alert(text)

Sends a simple message to tray
